# Wukong Suffering Tracker

A personal death-and-kill tracker for every boss in *Black Myth: Wukong*. Built with Vite + React + TypeScript, styled to feel like an artifact from the game world.

## Local dev

```bash
npm install
npm run dev      # http://localhost:5173
npm run build
npm run preview
```

## Stack

- Vite 6 + React 19 + TypeScript
- Tailwind CSS with custom design tokens
- Zustand (state + localStorage persistence)
- Tenor API (reaction GIFs)
- lz-string (shareable URL compression)
- GitHub Pages (hosting, free forever)
